Exemplo 11 CSS - Box Model

Box Model

O modelo de caixa(Box Model) do css é um conceito fundamental que define como os elementos HTML são representados coma caixas retangulares do documento HTML. Essa representação permite o controle preciso dos layouts e espaçamentos.

Componentes do Box Model

  1. Conteúdo(Content) área onde o conteúdo, como texto, imagens, etc, serão exibidos. Suas dimensões são definidas pelas propriedades height e width
  2. Padding(Preenchimento) Espaço invisível entre o conteúdo e a borda deste conteúdo.
  3. Border(Borda) linha que circunda o contéudo e o preenchimento(padding), podendo ter expessura, estilo e cores definidos.
  4. Margin(Margem) Espaço transparente externo à borda utilizado para separar o elemento dos outros elementos do documento html
