Tag <main>

O elemento main deve ser usado uma vez por página e deve envolver o conteúdo principal exclusivo da página.

Web Semântica

A web semântica é uma abordagem que utiliza elementos html com significado claro para estruturar conteúdo na web. São exemplos de tags semânticas <main>, <header>, <nav>, <section>, <article>, <aside>, <footer>

Eu posso ter em um documento html várias seções.

Elemento Semântico <div>

A tag HTML div é um elemento semantico com valor neutro. Ele é muito utilizado para organizar conteúdo e possui valor semântico neutro ou nulo.

Seletores CSS

Os seletores CSS, são padrões usados para selecionar e estilizar elementos hmtl. Junto com as propriedades e valores eles formam as regras CSS.

Seletor de Tipo

Os seletores de tipo, são aqueles que representam as tags html, por exemplo: h1{propriedade:valor};

Seletor de Id

O seletor de Id é aquele que utiliza o atributo html id. Esse seletor é personalissimo e deve ser aplicado uma vez que em tese o elemento html só terá um id que em tese não pode se repetir. O seletor de id deve começar com o #nomeSeletor{propriedade:valor};

Seletor de Classe

O seletor de classe é aquele que aplicado para estilizar um ou mais elementos hmtl que possuam o mesmo atributo class. Esse seletor começa com um ponto seguido do nome do seleto. Por exemplo .nomeSeletor{propriedade:valor};

Tag Semântica Aside

A tag semântica aside é aquela usada para representar conteúdo que está relacionado ao conteúdo principal, ou seja é um conteúdo auxiliar. É um conteúdo extra. Geralmente é usado também como menu auxiliar.

Elemento Semântico Footer

O elemento footer é o elemento semântico que finaliza o documento. Nele geralmente teremos as informações finais do seu documento. Por exemplo:
