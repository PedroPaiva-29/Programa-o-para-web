Vinculação Interna - Incorporada e Inline

A vinculação inline é uma forma de aplicar estilos css diretamente a um elemento html especifico, usando o atributo style. Diferentemente da vinculação externa esse tipo de vinculação ocorre diretamente com elemento.

Esse tipo de vinculação é pouco recomendado, pois dificulta a manutenção do documento html.

Nesta div estamos aplicando a vinculação inline
