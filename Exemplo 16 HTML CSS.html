O que é o Grid Layout

O Grid Layout é bidimensional, permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do FlexBox que é unidimensional, ordenando os elementos em linha.

FlexBox ou Grid Layout

O FlexBox e o Grid Layout podem ser usados em conjunto. Você pode aplicar o FlexBox a elementos dentro de um Grid Layout, aproveitando o melhor dos dois mundos.

A escolha entre FlexBox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu Layout e escolher a técnica que melhor atende a essas necessidades.
